#git 
- Tools : 
```sh
wget https://github.com/arthaud/git-dumper
```

- Run git dumper : 
```sh
./git_dumper.py http://website.com/.git ~/website
```

- Recover files : 
```sh
git status 
git restore --staged <files>
git restore <file>
```

- Get logs :
```sh
git checkout
git log
git show a8673b295eca6a4fa820706d5f809f1a8b49fcba`
```
